# 🚗 Autonomous Driving

This directory houses all materials related to the Autonomous Driving course, divided into two main macro-areas: **Control** and **Perception**.

## 📂 Structure

### ⚙️ [Control Material](./control_material/)
Focuses on vehicle dynamics, trajectory planning, and control algorithms.
*   **`assignments/`**: Python/C++ implementations of various controllers (e.g., Pure Pursuit, Stanley Controller, Model Predictive Control - MPC).
*   **`exercises/`**: Practice scripts and simulations.
*   **`support_material/`**: Additional documentation and references for vehicle models.

### 👁️ [Perception Material](./perception_material/)
Focuses on computer vision, point cloud processing, and state estimation.
*   **`assignments/`**: Practical projects involving Lidar data processing, object tracking, and filtering.
*   **`exercises/`**: 
    *   `1_pcl_exercises`: Exercises utilizing the Point Cloud Library (filtering, segmentation, clustering).
    *   `2_kalman_filter`: 1D and 2D Kalman Filter implementations for sensor fusion and tracking.
    *   `3_localization`: Particle Filter and SLAM concepts.
