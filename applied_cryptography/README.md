# 🔐 Applied Cryptography

This directory contains study materials and resources for the Applied Cryptography course.

## 📄 Contents

*   **`algorithm_questions.pdf`**: A collection of common exam questions and theoretical queries covering various cryptographic algorithms.
*   **`cryptography_algorithms_summary_23_24.pdf`**: Comprehensive summary notes on cryptographic algorithms, focusing on the 2023/2024 academic year syllabus. Topics include symmetric/asymmetric encryption, hashing, digital signatures, and key exchange protocols.
