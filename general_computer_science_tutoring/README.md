# 👨‍🏫 General Computer Science Tutoring

This directory holds a structured series of presentations and study materials created for tutoring introductory computer science concepts. It is an excellent resource for learning fundamental programming logic and Python concepts.

## 📚 Curriculum

*   **`03_boolean_algebra_binary_logic.pdf`**: Introduction to binary systems and Boolean logic.
*   **`04_conditional_statements_if_else.pdf`**: Control flow and decision making.
*   **`05_iterative_constructs_while.pdf`**: Looping mechanisms (while loops).
*   **`06_functions_standard_libraries.pdf`**: Introduction to reusable code and standard libraries.
*   **`07_functions_definition_strings.pdf`**: Deep dive into function creation and basic string manipulation.
*   **`08_strings_lists_tuples.pdf`**: Essential Python data structures.
*   **`09_vectors_matrices_for_loop.pdf`**: Advanced looping (for loops) and multidimensional data.
*   **`10_matrices_in_depth.pdf`**: Advanced matrix operations and algorithms.
*   **`11_dictionaries_file_management.pdf`**: Key-value mappings and I/O file handling.
