# 🌐 Distributed Systems

This directory contains theoretical summaries, exam preparation materials, and the final project for the Distributed Systems course.

## 📄 Theoretical Materials

*   **`distributed_algorithms_complete_theory.pdf`**: An extensive document covering the theory of distributed algorithms, including consensus, logical clocks, mutual exclusion, and fault tolerance.

## 💻 DTIP (Distributed Threat Intelligence Platform)

The **`dtip/`** folder contains the final project for the course. DTIP is a distributed system designed for threat intelligence.
*   **`dtip_project/`**: The main source code, utilizing Java for the core nodes and Python for the dashboard/TUI.
*   **Documentation (`.md` files)**:
    *   `architecture.md`: System architecture overview.
    *   `code_flow.md`: Explanation of execution flow and component interaction.
    *   `dtip_complete_guide.md`: The complete manual to setup and run the platform.
    *   `project_theory_study.md` & `theory_reference.md`: Theoretical foundations applied to the project.
    *   `uml_protocols.md`: UML diagrams modeling the distributed communication.
    *   `exam_speech.md`: Presentation notes for the project defense.
